Add README.md for Hexo

Github上一个项目总是会有个README.md的介绍文档，但是Hexo在部署文章时会渲染*.md文件，我的解决方案如下：

在\source目录下添加README.md文件。

在全局config.yml文件中找到skip_render,修改为：

skip_render: README.md

然后在GitBash中重新部署。

hexo clean
hexo generate
hexo deploy

然后在目录中可以看到README.md文件成功添加上。
